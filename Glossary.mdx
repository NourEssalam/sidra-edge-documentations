# Glossary

## Deployer

A user who deploys projects on the Sidra Edge platform. Deployers can upload their own projects or choose from a variety of templates.

## Provider

A user who offers computing resources through Sidra Edge. Providers can set pricing, manage resources, and receive payments from deployers.

## Ethereum

A decentralized cryptocurrency used for transactions on the Sidra Edge platform. Ethereum provides a secure and transparent payment system for all users.

## Edge Computing

A distributed computing paradigm where computation is performed closer to the data source, reducing latency and improving performance. Sidra Edge utilizes edge computing to deliver fast and efficient deployment solutions.
